- Plugin Users
Like most extensions requiring no registration, 'ClickBait Filtering Plugin' collects non-personally-identifying information of the sort that web browsers and servers typically make available, such as the browser type, language preference, referring site, and the date and time of each user request. 'ClickBait Filtering Plugin's purpose in collecting non-personally identifying information is to better understand how it's clients use the Plugin and improve the functionality of the Plugin itself. Regardless 'ClickBait Filtering Plugin' may never release non-personally-identifying information in any format being aggregate or other, e.g., by publishing reports or findings.

'ClickBait Filtering Plugin' also collects potentially personally-identifying information like Internet Protocol (IP) addresses of users. These addresses are stored as a random hash added to non-personally-identifying information and are not retrievable by any means. Nevertheless, 'ClickBait Filtering Plugin' never discloses user IP addresses under any circumstances.


- Gathering of Personally-Identifying Information
All users of this Plugin choosing to interact with 'ClickBait Filtering Plugin' exist as an entity in the Plugin database by a combination of the abovementioned browser features and HTTP request information. The amount and type of information that 'ClickBait Filtering Plugin' gathers in order to create a unique user profile, depends on the nature of the interaction, but is always limited to the abovementioned features.

- Tokens
To enrich and perfect your online experience, 'ClickBait Filtering Plugin' uses 'JWT Tokens'. Similar technologies and services are provided by others to display personalized content, and store your preferences on your own computer.

A Token is a string of information that a web applications stores on a users’ device, and that the users’ application (our plugin in this case) provides to the Plugin Server API each time the user returns to using it. 'ClickBait Filtering Plugin' uses Tokens to help 'ClickBait Filtering Plugin' identify and track user, their usage of the Plugin, and their website access preferences. 'ClickBait Filtering Plugin' users who do not wish to have Tokens placed on their computers should restrain from using the Plugin. By continuing to use the Plugin, you hereby acknowledge and agree to 'ClickBait Filtering Plugin's use of Tokens.
